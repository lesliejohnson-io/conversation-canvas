Product Context

Paloma Session Canvas captures meaningful phrases from a coaching conversation and places them on a map showing:

Past ↔ Future
Troubled ↔ Resourceful

This creates a visual structure that reveals where the conversation is happening psychologically.

The system is human-in-the-loop.
The coach can override classifications when needed.

The goal is cognitive clarity at a glance.

Dialogic Map (Primary Visualization)

The left panel contains the Dialogic Map.

This is a quadrant visualization with crosshair axes.

Axes should look clean and precise, not decorative.

Horizontal axis:

Past ← → Future

Vertical axis:

Troubled ↓ ↑ Resourceful

Add small arrowheads at axis ends to show directional meaning.

The map background should remain mostly white.

Each quadrant should have extremely subtle tinting for orientation.

Example palette:

Preferred Future → soft green tint
Resourceful Past → soft blue tint
Troubled Past → soft amber tint
Dreaded Future → soft rose tint

These colors should be barely noticeable, just enough to guide the eye.

Recency Structure

The map contains 5 faint concentric rings radiating outward from the center.

Purpose:

Represent recency of statements.

Center = most recent phrases

Outer rings = earlier phrases

Rings should be extremely subtle.

Phrase Nodes

Each extracted phrase appears as a node on the map.

Node design:

Small circular dot

Number label

Short phrase text

Example:

3
"confident leader"

Phrase labels should be limited to 4 words.

Visual Encoding

Node size = importance
Node opacity = recency

Older phrases become slightly lighter.

Nodes closer to center feel more visually prominent.

Hover Interaction

When hovering a node:

Node enlarges slightly

Phrase label becomes slightly bolder

Corresponding item in the side panel highlights.

Click Interaction

When clicking a node:

Node becomes selected.

A small floating quadrant override menu appears.

Options:

Resourceful Past
Preferred Future
Troubled Past
Dreaded Future

These appear as small pill chips.

Side Panel (Captured Phrases)

Right panel contains the ranked phrase list.

Each row includes:

Number
Phrase label
Quadrant tag
Source sentence reference

Example:

3 — confident leader
Preferred Future
Sentence 12

The list should feel structured and scannable.

Spacing between rows should be comfortable.

Synchronization

Hovering a node highlights the corresponding phrase row.

Hovering a phrase row highlights the corresponding node.

Selected node should remain highlighted until another is selected.
